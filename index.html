PORTFOLIO

Trending Plays - Web App

www.trendingplays.com
Github Repo

❤ What is it? Web-app created using Python/Flask, HTML, CSS & Docker

❤ What does it do? Utilizes Twitter API v2 to display the top 10 trending stocks on Twitter.

❤ How does it work? An event scheduler is set up to run a script each week that pulls the weekly tweet count for each ticker, updating it in the database. The web app, in turn, pulls and displays the most recent top 10 counts from the database upon each visit.

❤ Upcoming Features! User will be able to click each ticker to view an informational page with basic stock info as well as see recent news.

Calendar App

Github Repo

❤ What is it? Web-app created using Python/Flask, HTML, CSS, JS & MySQL

❤ What does it do? Lets user add appointments to their calendar and displays daily appointments.

❤ How does it work? Utilizes form to update database with user input to create appointments. The app also displays appointments according to the current date.

❤ Upcoming Features! User will be able to change the date to see appointments for that date. User will also be able to delete appointments.
